# Deferred Scope

These items are intentionally out of the current implementation slice:

- Advanced staff permissions and granular audit administration
- QR-first event scanning and wallet-style ticket flows
- Complex ticket tiers, promo codes, waitlists, and seat maps
- Self-service refunds and partial refund workflows
- Rich CRM, segmentation, loyalty, and advanced analytics
- Deep Square inventory sync, writeback, and order-level integration
- Multi-location corporate management and enterprise reporting

The repo includes route and schema scaffolding for later milestones, but the implemented UI is deliberately centered on venue setup, manual item CRUD, and multi-surface menu distribution.
